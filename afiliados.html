¿Cómo crear un Token de Acceso Personal?

  1. Ve a GitHub Settings → Developer Settings → Personal Access Tokens
  2. Haz clic en "Generate new token"
  3. Selecciona el scope repo (acceso completo a repositorios)
  4. Dale un nombre descriptivo (ej: "Sistema Afiliados Bateas")
  5. Haz clic en "Generate token"
  6. IMPORTANTE: Copia el token inmediatamente (solo se muestra una vez)
  7. Pega el token en el campo de arriba
